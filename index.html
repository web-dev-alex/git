install git

1. öffne visual studio code

2. auf der linken seite befindet sich eine vertikale bar, in der das source control icon angewählt wird

3. dann öffnet sich ein neues fenster, welches den direkten link zur installation von git bereit hält

4. auf der externen seite installieren und visual studio neu starten

5. für die folgenden befehle kann das terminal in visual studio code verwendet werden



basisfunktionen in der richtigen Reihenfolge

1. im terminal das verzeichnis der gewünschten datei auswählen

2. dann mit dem Befehl: git init leeres repository initialisieren

3. mit dem Befehl: git statuswird eine übersicht von verfolgten/nicht verfolgten dateien gezeigt
